# iso-week [![NPM version](https://badge.fury.io/js/iso-week.svg)](http://badge.fury.io/js/iso-week)

> Get the ISO week of the year.

## Install with [npm](npmjs.org)

```bash
npm i iso-week --save
```

## Run tests

```bash
npm test
```

## Usage

```js
var isoWeek = require('iso-week');

isoWeek();
//=> '50'
```
